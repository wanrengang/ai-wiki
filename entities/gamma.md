# Gamma

> AI Native 演示文稿生成平台，通过自然语言输入生成精美 PPT。

## 概述

Gamma 是一个 AI Native 演示文稿生成平台，让用户通过自然语言输入就能生成精美的 PPT，无需手动设计排版。核心理念是"像和朋友聊天一样做 PPT"。

## 核心功能

| 功能 | 说明 |
|------|------|
| 文字转 PPT | 自然语言描述→自动生成 15+ 页精美 PPT |
| 智能设计 | 配色方案、字体搭配、布局排版、图片匹配、图表生成 |
| 多模态输入 | 文字、Word/PDF 上传、网页链接、图片素材 |
| 实时协作 | 多人编辑、评论反馈、版本历史、分享链接 |
| 一键导出 | PDF、PowerPoint、图片、在线分享链接 |

## 效率对比

| 工具 | 总耗时 |
|------|--------|
| 传统 PowerPoint | ~5 小时 |
| **Gamma** | **~30 分钟** |

效率提升：**10 倍**

## 定价

| 版本 | 价格 | 功能 |
|------|------|------|
| 免费版 | $0 | 400分数/月、基础功能、带水印 |
| Pro 版 | ~$10/月 | 无限分数、无水印、导出 PPT、高级模板、品牌定制 |
| 企业版 | 定制 | 团队协作、管理后台、私有部署 |
