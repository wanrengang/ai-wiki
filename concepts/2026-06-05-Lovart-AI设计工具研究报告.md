## 产品概述

**Lovart** 是一个 T0 级 AI 设计 Agent，定位为"随时待命的专业设计师"，适用于自媒体内容创作、电商视觉设计、品牌视觉管理等多个场景。

## 核心功能

### 1. Brand Kit（品牌套件）

- 上传品牌文件（PDF）自动识别 Logo、颜色、字体构建品牌资产
- 支持纯设计指南构建，灵活度最高
- 实际用法：输入一套设计指南（如"OpenAI 关键词用 OpenAI 风格，Claude 关键词用 Anthropic 风格"），后续创作直接 @ 调用，无需重复描述

**作者的设计指南示例：**
```
遇到关键词为OpenAI（包括ChatGPT，codex）时，采用OpenAI的品牌设计风格
遇到关键词anthropic（包括claude code，claude），采用anthropic的品牌设计风格
遇到关键词谷歌（包括DeepMind）时，采用谷歌品牌设计风格
一般采用类似苹果的极简设计风格
设计哲学：色彩搭配错了画面瞬间变丑 —— 保护美学比给自由更重要
字体系统：标题必须是衬线，正文必须是非衬线，元数据必须是等宽
英文单词在中文标题中渲染为 Playfair Display 斜体
```

### 2. Font Generator（字体生成器）

- 上传图片或文字描述 → 生成一整套字体
- 生成后自动存入「My Fonts」，随时复用，无需安装
- 与 Brand Kit 搭配，字体也能纳入品牌规范

### 3. Create Skill（工作流保存）

- 把一次满意的创作对话一键保存为可复用 Skill
- 下次遇到同类项目直接调用，不用重新想提示词
- 解决"同一套流程每次都要重新跑，结果不稳定"的问题

**核心价值：稳定复现一个好结果，比偶尔出一张特别好的图重要得多。**

### 4. Export PSD（接 PS 的口子）

- 将画布内容导出为带图层的 PSD 文件
- 保留完整图层结构，打开 Photoshop 即可继续精修
- 适合"Lovart 出图 + PS 精修"的工作流

### 5. Agent 能力

- 理解自然语言创意描述，自动补充细节
- 行业新模型出来后第一时间接入
- 无限画布支持多版本对比和局部修改
- 出图质量极其稳定，达到专业设计师交付水准

## 电商实战案例（618 工作流）

作者演示了一套美妆品牌 618 大促素材的全流程：

### 步骤一：淘宝主图 4 张
Prompt 示例：
> 为我设计一套淘宝主图，唇釉logo，四只唇釉【商品款式1-4图】。一张主图、一张卖点图、一张色号展示图、一张四宫格四色上唇图（同角度）。画幅1：1。风格：排版高级、50%产品+20%文字+30%负空间、全中文。对比强烈，深色文字/图案、高明度灰色背景，思源黑体Light&Medium。

### 步骤二：618 活动主图
> 参考【风格参考图1】的排版，生成一张超级618的活动主图，画幅1:1，背景色高明度灰。需要包含：天猫618活动logo、核心卖点标题、产品名字、色号展示，价格：到手价170元/支，优惠信息：第二件半价。

### 步骤三：赠品图
> 参考【风格参考图2】的排版，为品牌生成一张赠品图。内容需要包含：买任意3支唇釉赠送热门腮红棒、卖点、限时赠送、小字说明（活动解释权、活动规则、时间）

### 步骤四：商详图
> 根据主图生成一套详情图，宽度750px，内容包括：一张618活动主图、一张hero图（模特、面向观众、手拿产品）、一张卖点图、4张色号展示图
