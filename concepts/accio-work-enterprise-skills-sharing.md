# Accio Work 企业版：团队 Skills 共享功能

> 来源：微信公众号「数字生命卡兹克」
> 日期：2026年5月26日

## 背景

作者公司一直头疼如何让 Skills 在团队内流通——最低成本进行团队内部的 Skills 同步。自己被迫 vibe coding 了一个 Skills Hub 给公司内部用，已有 19 人共享了几十个 Skills。

痛点：共享方式原始（压缩包/GitHub链接），版本混乱，更新麻烦。

## 核心功能

### Skills 同步

- 上传到团队工作空间
- 管理员可设置「全员可见」（共享）
- 成员端显示「未安装」的团队 Skill，点加号即可安装
- 更新：作者更新后，成员点「一键更新」即可同步

### Agent 共享

- 成员可推荐自己的 Agent 给团队
- 普通成员推荐的 Agent 默认不可见，需管理员勾选「所有人可见」
- 共享后其他成员可直接与该 Agent 对话

### 团队管理

- 成员邀请：邮箱邀请，分配积分点数
- 权限分级：所有者 > 管理员 > 普通成员
- 版本档位：5人/15人/50人团队

## 关键观点

> "Agent产品下一个战场，就在团队协同这一层"

- 单兵作战时代快过去了
- 团队能否自然共享能力，是被忽视的刚需
- Accio Work 总是「引领版本」——多Agent协同、团队共享都先做

## 技术借鉴

| 功能 | 实现思路 |
|------|---------|
| Skills 版本同步 | 中心化存储 + 一键更新 |
| 权限控制 | 三级权限 + 管理员审批 |
| 自动更新 | 轮询机制或登录时 Check |
