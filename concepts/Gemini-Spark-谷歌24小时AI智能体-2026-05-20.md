# Gemini Spark：谷歌 24/7 个人 AI 智能体研究报告

## 概述

Gemini Spark 是 Google 在 I/O 2026 大会上发布的**24/7 个人 AI Agent**，核心定位是将 Gemini 从"对话式 AI 助手"升级为"能自主执行任务的主动型数字伙伴"。

**核心本质**：一句话让 AI 替你干几份活，不用担心乱搞数据。设备关机、锁屏甚至断网，Spark 仍在云端持续工作。

---

## 技术架构

| 组件 | 说明 |
|------|------|
| 底层模型 | Gemini 3.5 Flash（同日发布，代号"卡布奇诺"） |
| Agent 框架 | Google Antigravity |
| 运行载体 | Google Cloud 专属虚拟机（不依赖用户设备） |
| 协议支持 | MCP（Model Context Protocol） |
| 第三方集成 | Canva、OpenTable、Instacart（首批）等 |

**与 OpenClaw 的对比**：
> Gemini Spark 就是谷歌官方闭源、全托管、且加了超级 Buff 的"商业版 OpenClaw"。它直接调用谷歌全家桶的私有底层 API，不需要像 OpenClaw 那样苦哈哈地解析网页 HTML。目前完成度最高、商业化最成熟的"全自动挂机搬砖智能体"。

## 核心功能

### 1. 24/7 后台持续运行
- 运行在 Google Cloud 专属虚拟机上
- 手机锁屏、笔记本关机、断网状态下都在云端持续工作
- 每个用户获得一个专属 Spark Gmail 地址

### 2. 任务（Tasks）
用自然语言下达单次或多步骤任务，Spark 自动跨 Gmail、Calendar、Drive、Docs 等应用执行。
- 示例："帮我检查本月信用卡账单里有没有隐藏的新订阅费用"

### 3. 技能（Skills）—— 可学习
用户展示一次如何完成任务，Spark 学习后可重复执行。
- 示例："每天早上从孩子学校的邮件里提取重要截止日期，发给我和我老婆"

### 4. 定时任务（Schedules）
设定定时触发规则，按周期自动执行重复性任务。
- 示例："每月 1 号扫描信用卡账单里的隐藏费用"

### 5. 多线程并行
可同时运行多个任务线程，并行处理不同工作流。

### 6. Google Workspace 深度集成
原生连接 Gmail、Calendar、Drive、Docs、Sheets、Slides、YouTube、Google Maps。所有连接默认关闭，用户在设置中主动开启。

### 7. 第三方 MCP 集成
通过 MCP 协议连接第三方服务（Canva、OpenTable、Instacart 等）。

### 8. 安全确认机制
涉及高风险操作时会请求用户确认后再执行，所有 Workspace 连接默认关闭。

### 9. 跨设备无缝切换
手机启动任务 → 电脑查看进度 → 任何设备对话，状态全程同步。

## Karpathy 加入 Anthropic 的关联

**第二件大事**：AI 圈传奇研究员 Karpathy 正式加入 Anthropic，加入的是**预训练团队**而非当时最火的 Agent 方向。

**判断**：模型本身还有大的提升空间，Karpathy 选择把模型本体能力再往前推一步。AI 模型能力还在加速进化，这个趋势至少还要再持续两三年。
