# MiroFish：群体智能预测引擎

> 来源：微信公众号「5G创见」，作者：九宝，2026-05-31

## 概述

**MiroFish** 是一款开源群体智能预测引擎，由盛大集团孵化，2026年3月开源后在 GitHub 迅速斩获 **63.2k Star**，成为现象级 AI 项目。

## 核心流程

```
输入种子材料（新闻/政策/小说）
  ↓
MiroFish 自动提取信息，构建高保真平行数字世界
  ↓
生成数千个具备独立人格、长期记忆、行为逻辑的 AI 智能体
  ↓
智能体自由交互、社会演化
  ↓
输出结构化预测报告
```

## 三大颠覆点

| 特点 | 说明 |
|------|------|
| **零门槛操作** | 用自然语言描述场景，全流程自动完成，无需写代码 |
| **群体智能** | 数千智能体产生涌现性社会行为，单个大模型无法模拟 |
| **动态干预** | 模拟中可随时注入新变量，实时观察反馈 |

## 技术栈

- **模拟引擎**：CAMEL-AI OASIS — 处理大规模智能体社会模拟
- **知识图谱**：GraphRAG — 自动从种子文档提取实体关系
- **记忆管理**：Zep Cloud — 智能体长期记忆管理
- **大模型**：兼容 OpenAI SDK，官方推荐阿里云百炼 qwen-plus

## 应用场景

- **舆情危机管理**：模拟新闻事件、品牌危机在社交群体中的传播路径
- **金融场景预测**：建模财报发布、监管变化后的市场情绪动态
- **政策影响测试**：零风险测试公共政策草案，模拟不同群体反应
- **创意叙事探索**：如基于《红楼梦》前80回推演后四十回结局

## Agent 新范式

MiroFish 揭示了四个趋势：

1. **从「对话」到「模拟」** — ChatGPT 被动响应，MiroFish 主动演化
2. **从「单智能体」到「群体智能」** — 群体智能是 Agent 研究的下一个深水区
3. **从「闭源产品」到「开源基础设施」** — AGPL-3.0 开源，Docker 一键部署
4. **从「专业机构专属」到「普通人可用」** — AI 民主化加速

## 使用方式

- **在线体验**：mirofish.work
- **本地部署**：Node.js ≥18，Python ≥3.11且≤3.12，执行 `npm run setup:all` && `npm run dev`
- **Docker 部署**：`docker compose up -d`
