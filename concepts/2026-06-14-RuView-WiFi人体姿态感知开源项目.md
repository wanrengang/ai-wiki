# RuView: WiFi空间感知技术

## 概述

RuView (github.com/ruvnet/RuView) 是一个利用商品WiFi信号实现实时空间感知、生命体征监测的开源项目。GitHub星标73.7k，9.8k fork。

核心技术基于卡内基梅隆大学DensePose From WiFi研究，通过CSI (信道状态信息) 捕捉WiFi信号穿过人体时的细微变化，无需摄像头即可实现人体姿态估计。

## 技术原理

### CSI信道状态信息
- 捕捉56个子载波的幅度和相位变化
- 生成人体"信号指纹"
- 原本用于优化无线传输质量

### WiFlow架构
- TCN (时序卷积网络): 处理CSI时间序列
- 轴向注意力: 提取空间特征
- 输出17个COCO关键点 (鼻子/肩膀/手肘/膝盖等)
- 精度: 92.9% PCK@20

### 模型规模
- 原始: 1.8M参数
- 4bit量化: 881KB
- 可在ESP32 ($9) 上实时运行

## 核心能力

| 功能 | 指标 |
|------|------|
| 人体存在检测 | 82.3% accuracy (已修正) |
| 呼吸监测 | 6-30 BPM |
| 心率监测 | 40-120 BPM |
| 穿墙感知 | 5米水泥墙 |
| 推理延迟 | <10ms |
| WASM边缘模块 | 65个, 5-30KB each |

## 隐私设计

- 所有处理本地完成，不上传云端
- 不存储图像，不识别身份
- Ed25519加密见证链
- 代码开源可验证

## 智能家居集成

- Home Assistant (MQTT)
- Apple Home / HomePod (HAP-1.1)
- Google Home / Alexa
- SmartThings (Matter)

每节点21个实体，含10种语义状态（有人睡眠/跌倒风险/房间活跃等）

## 应用场景

1. **灾难救援**: 穿透废墟探测被困人员
2. **智能家居**: 人体位置感知自动控制
3. **养老监护**: 跌倒检测、长期不活动告警
4. **睡眠监测**: 无接触生命体征追踪
