# AIFUT 大会

## 概述

AIFUT（AI Future）是中国人工智能领域的重要大会，聚焦 AI 技术前沿与产业落地。大会主题常围绕"理性+感性，人文与科技的十字路口"展开。

## 关键事实

| 项目 | 内容 |
|------|------|
| 地点 | 北京亦庄电竞馆 |
| 规模 | 1500+ 参会者，3500 平米展区 |
| 核心主题 | 人工智能与人文科技融合 |

## 主要嘉宾与观点

### 王磊 - 亦庄的 AI 三层实践
- **产业层**：AI 重构产业底层逻辑，开放 100 个场景
- **城市层**：建设全域人工智能之城，600 平方公里车路云一体化
- **人层**：推出"模数 OPC 社区"，服务超级个体

### 卡兹克
> "唯一不能被重新定义的，是好奇心"

- AI 吞噬软件（2011 年是软件吞噬世界）
- GUI 时代人操作软件 → Agent 时代委托 Agent 干活
- 飞书现在为 Agent 而生，CLI 已开源

### 罗振宇
> "未来人类只剩一件事——告诉 AI 你要什么"

- 人最后的稀缺性 = 你想要什么
- 三级台阶：独特性 → 丰富性 → 魅力

### 张鹏 - 最适合 Agent 的大模型
- 定义问题能力是核心
- 国内外差距：6 个月到 1 年
- 从人的第一性出发，打开想象力

### 谷大白话
> "AI 时代的教育是纯享的巧克力"

- AI 让教育变成纯享（游戏化三段式最终形态）
- 积累 200 万条真实亲子对话数据
