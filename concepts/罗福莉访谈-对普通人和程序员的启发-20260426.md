# 罗福莉访谈 — 对普通人和程序员的启发

## 访谈背景

天才少女罗福莉（DeepSeek-Lumiina 项目核心成员）的访谈整理，原文 3.5 小时访谈内容精编。

---

## 对普通人的核心观点

1. **Agent 工具正在快速替代 80% 的日常工作** — 现在不学 Claude Code/OpenClaw 就有落后风险
2. **成本断崖式下降** — 10元 API 费 + 好的 Agent 框架 = 以前 1000 元的工作
3. **一人公司时代来临** — Multi-Agent 协作撑起微型企业的现实已来
4. **经验有价值** — 沉淀为 Skills 是 Agent 无法自学到的"另类信息"
5. **AGI 不是某个时刻** — 而是生活方式被慢慢改变，现在就参与

---

## 对程序员的启发

### 核心转变

| 旧范式 | 新范式 |
|---|---|
| 写代码的人 | 架构设计者 + 需求澄清者 |
| 技术栈经验 | 对数据分布的直觉 + 模型能力边界的感知 |
| 刷八股算法 | 思维方式和底层原理 + 大模型应用探索 |

### 关键判断

- **85% 日常编程**已可由中层模型 + Agent 框架完成
- **Code 是最强泛化力** — 程序员是 AI 时代第一受益人
- **不要读论文了，亲自用工具** — 花一晚装 OpenClaw 比读 10 篇论文更有价值
- **未来决定性能力**：快速适应 Agent + 模型协作范式，培养"数字分身"

### 职业方向建议

- **Foundation Model**：1T 参数是入场券，卡资源是瓶颈
- **AI Infra**：RL Infra 需求 > Pre-train Infra（需容忍模糊性）
- **应用层**：一人公司 + Agent 框架 = 以前 10 人团队的工作

> "不需要所有人都去训模型，但所有人都应该学会用 Agent 重新定义自己的工作方式。"
