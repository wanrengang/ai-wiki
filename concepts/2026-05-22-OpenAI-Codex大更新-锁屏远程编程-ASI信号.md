# 今天，OpenAI Codex大更新！锁屏也能隔空打工，奥特曼强推

> 来源：微信公众号「新智元」
> 副标题：ASI启示录
> 日期：2026-05-22

## 核心更新：五大能力

### 1. Appshots：双击 Command 一键读懂所有屏

操作极其简单，只需 `Command+Command`，AI 就能瞬间「看穿」屏幕。不仅能读取窗口中没有滚动到的文本，还能提取文件路径、URL 等隐藏信息，直接编码。

### 2. /goal 自主编码正式毕业

无论任务大小，现在所有 Codex 开发者可以解锁 AI「自动编码」能力。奥特曼宣布：每当 Codex 新增 100 万周活用户，就会重置一次所有用户的使用限额（「每增长100万就重置，直到1000万为止」）。

### 3. Mac 锁屏远程编程

最令人震惊的能力：如今的 Codex 可以在 Mac 休眠状态下，无需解锁，7×24 小时无限编程。AI 能在锁屏状态下隔空打工。

### 4. ChatGPT 杀入 PowerPoint

同一天，ChatGPT 以插件形式正式杀入 PowerPoint。Beta 版今天上线，一句话就能从 0 开始生成一整套可编辑的 PPT，全网打工人再次破防。

### 5. 全栈工作平台蜕变

Codex 正在从一个「编程助手」，蜕变为一个全栈工作平台。
