## Cursor

**本周有重大更新**

### 版本与功能更新

- **Bugbot 全面升级**：审查速度提升 3 倍（平均耗时从 ~5 分钟降至 ~90 秒），成本降低 22%，发现 bug 数量提升 10%（0.56 → 0.62/每次 review）。由新版 Composer 2.5 驱动。
- **发布 `/review` 命令**：支持在 push 前运行 Bugbot 和 Security Review，自动识别已审查过的 PR diff 并跳过重复审查（支持 GitHub & GitLab）。
- **Bugbot 增量审查**：可配置仅审查自上次审查以来的新增内容，保持反馈聚焦。
- **SDK 批量更新**（TypeScript & Python）：
  - **Custom Tools**：无需自建 MCP 服务器，通过函数定义即可向 Agent 暴露自定义工具，且子 Agent 可继承使用。
  - **Auto-review**：headless 模式下可将工具调用路由至自动审查分类器，用自然语言配置 `permissions.json` 控制放行/拦截策略。
  - **JSONL Store**：元数据存储支持 JSONL 格式（append-only，可读可 diff 可入版本控制），替代原有的 SQLite。
  - **Nested Subagents**：子 Agent 可递归生成子 Agent，各层级保持独立 prompt 和模型。
- **Canvas Design Mode**：支持在 Canvas 中直接点击、绘制、语音标注 UI 元素来指导 Agent 修改；新增 Context Usage Report 可视化 token 消耗分布。

### 社区热点
- Cursor 3.7+ 版本持续获得开发者关注，Bugbot 升级被社区评为"本季度最有价值的 Agent 改进"。

## Codex（OpenAI）

**本周有产品信息更新**

### 产品定位更新

- OpenAI 官网 Codex 页面明确为"AI Coding Agent"，定位为端到端完成工程任务（功能开发、复杂重构、迁移等）。
- 核心能力：
  - **Multi-agent 工作流**：内置 worktrees 和云端环境，支持多 Agent 并行工作。
  - **Skills**：超越代码生成，直接参与代码理解、原型设计、文档等工程任务。
  - **Automations**：持续后台运行，自动处理 issue 分流、告警监控、CI/CD 等日常事务。
  - **代码审查**：强调在代码审查中能发现其他 bot 遗漏的 bug。
- 客户案例：Harvey（移动 lead）报告早期迭代效率提升 30–50%；Ramp 团队将其作为代码审查主力。

### 社区热点
