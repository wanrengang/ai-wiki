# Codex 视频创作 Skill 场景全拆解

## 概述

本文按实际创作场景拆解 Codex 视频制作技能，帮助创作者根据自身痛点选择正确的工具。

**核心观点**: 不要问"哪个 Skill 最强大"，要问"我的创作卡在哪个环节"。

---

## 六大场景与推荐 Skill

| 场景 | 痛点 | 推荐 Skill | 底层逻辑 |
|------|------|-----------|----------|
| 只有文案，没有素材 | 零基础起步 | **Remotion-based** | HTML/CSS动画渲染成MP4 |
| 批量化系列出片 | 数据矩阵重复 | **Motion/Remotion Template** | 代码模板，更换数据批量生成 |
| 剪辑口播素材 | 删口误/停顿/加字幕 | **Video Cut Skills** | 转录→识别卡顿→审核→自动剪辑 |
| 多素材复杂剪辑 | Vlog/生活记录散镜头 | **Video Utils / Juicer** | 删停顿/调色/加字幕/动态图表 |
| 缺辅助画面配乐 | B-roll/音乐缺失 | **Generative Media Skills** | AI临时素材库（图片/视频/白噪音） |
| 不会写视频提示词 | 不知道怎么生成成片 | **Scene-to-Video Skills** | AI导演，生成专业分镜Prompt |

## 场景详解

### 场景一：Remotion-based Skills
- **适用**: 纯图文转视频、概念科普类自媒体
- **示例**: 用 Remotion 做30秒竖屏视频，动态翻转卡片展示6个AI工具

### 场景二：Motion / Remotion Template Skills
- **适用**: 每周AI工具热度榜、行业数据周报等固定栏目
- **价值**: 模板化后一键批量生成几十条新视频

### 场景三：Video Cut Skills
- **适用**: 国内短视频创作者（已拍好口播素材）
- **流程**: 视频转录 → 识别卡顿静音 → 用户审核勾选 → 自动剪辑

### 场景四：Video Utils / Juicer Skills
- **适用**: 旅行Vlog、生活记录等已有散碎素材
- **功能**: 删停顿、调色、加字幕、嵌入动态图表
- **注意**: 通常需本地配置环境

### 场景五：Generative Media Skills
- **本质**: AI临时素材库，非剪辑工具
- **用途**: 补B-roll（辅助画面）、背景音乐、音效
- **注意**: 需调用大模型API，会产生实际费用

### 场景六：Scene-to-Video Skills
- **本质**: AI导演，不能直接生成视频
- **价值**: 生成专业分镜词（逐秒分镜+人物动作+镜头轨迹+对白+音效）
- **效果**: 复制到Luma/可灵/Runway，成片率提升数倍
