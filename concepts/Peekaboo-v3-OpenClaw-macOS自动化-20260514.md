# Peekaboo v3 — OpenClaw macOS UI 自动化

## 概述

2026年5月12日，OpenClaw 创始人 Peter 发布 Peekaboo v3，一天连更三个版本。Peekaboo 是 macOS UI 自动化 CLI，让 OpenClaw 拥有「视觉」和「触觉」。

## 核心能力

| 能力 | 说明 |
|---|---|
| 屏幕捕获 | 截取整个屏幕、特定应用窗口、UI 元素局部区域 |
| 元素定位 | 识别按钮、输入框、菜单项等 UI 元素，精确定位坐标 |
| 输入模拟 | 点击、双击、拖拽、滚动、键盘输入、快捷键；支持前后台应用 |
| 应用管理 | 启动应用、切换窗口、操作菜单栏、管理 Dock |

## 竞品优势

| 优势 | 说明 |
|---|---|
| 原生集成 | OpenClaw 官方 Skill，开箱即用 |
| 权限管理 | bridge host 处理权限申请，用户只需授权一次 |
| 快照缓存 | 多次操作复用屏幕状态 |
| 开源免费 | 无订阅费，无使用限制 |

**局限**：仅支持 macOS

## 意义

AI Agent 从「对话式」→「工具调用型」→「桌面自动化」的第三阶段跃迁。OpenClaw 从聊天助手进化为**能动手干活的数字员工**。
