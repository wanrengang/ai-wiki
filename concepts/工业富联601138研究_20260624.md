## 附录：术语解释

| 术语 | 含义 |
|------|------|
| EPS | 每股收益 = 净利润 ÷ 总股本 |
| YoY | 同比增长 = 与去年同期相比的增长百分比 |
| PE | 市盈率 = 股价 ÷ 每股收益 |
| H1/H2 | 上半年 / 下半年 |
| TTM | 滚动12个月 = 过去4个季度的数据累加 |
| 换手率 | 成交量 ÷ 流通股本 × 100% |
