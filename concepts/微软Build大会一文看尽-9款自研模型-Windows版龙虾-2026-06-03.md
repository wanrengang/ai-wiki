# 微软Build大会一文看尽：9款自研模型、Windows版龙虾、"梦中情机"，黄仁勋猛夸

> **来源：** 智东西（作者：陈骏达）  
> **发布日期：** 2026年6月3日 13:03  
> **原始链接：** https://mp.weixin.qq.com/s/F67rGgpZaKdPqxkinIpxLw

## 摘要

微软 Build 2026 大会全面解读：9 款自研 MAI 模型、Windows 版龙虾（OpenClaw）、"梦中情机"（Surface RTX Spark Dev Box），黄仁勋出席并猛夸合作。

## 核心内容

### MAI 模型家族（7款首发）

微软 AI CEO 穆斯塔法·苏莱曼发布 7 款自研 MAI 模型：

| 模型 | 类型 |
|------|------|
| MAI-Thinking-1 | 推理模型 |
| MAI-Image-2.5 | 文生图/图生图 |
| MAI-Transcribe 1.5 | 语音转写（43语言） |
| MAI-Voice-2 | 多语言语音合成 |
| MAI-Code-1 | 编程模型 |

### Windows 版龙虾（OpenClaw）

- OpenClaw 通过 MXC 原生运行在 Windows 上
- 系统安全隔离，企业可管控

### "梦中情机" — Surface RTX Spark Dev Box

- NVIDIA RTX Spark GPU
- 128GB 统一内存
- 1 PFLOPS AI 算力
- 预装 Windows 开发者优化系统

### 黄仁勋出席

NVIDIA CEO 黄仁勋出席 Build 2026，猛夸与微软的深度合作。
