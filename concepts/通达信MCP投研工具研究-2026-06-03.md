# 通达信MCP投研工具研究

> **研究日期：** 2026年06月03日  
> **用途：** AI投研工具调研

## 概述

**通达信MCP（Model Context Protocol）** 是通达信官方推出的 AI 行情数据接口服务，将通达信实时行情、财务数据、资金流向等核心数据通过 MCP 协议开放给 AI 助手，实现 AI 与金融数据的打通。

## 核心价值

让 AI 真正"看懂"市场，解决通用 AI 无法获取实时行情的痛点。

## 主要功能

### 1. 智能个股诊断
AI 自动调取：实时行情、技术指标（MACD/KDJ）、资金流向（北向资金）、核心财务数据（PE/PB/ROE），生成综合诊断报告。

### 2. 自然语言选股
用户用自然语言描述选股条件，AI 转化为 MCP 查询指令，快速返回符合条件的股票列表。

**示例：**
- "找出今天涨停、换手率低于5%、且北向资金净流入的股票"
- "PE小于20倍，ROE连续3年大于15%的医药股"

### 3. 产业链图谱分析
利用通达信产业链数据，AI 自动梳理上下游关系、代表个股，辅助投资逻辑分析。

### 工具分类（9大类）

| 分类 | 说明 |
|------|------|
| 连接管理 | 初始化/关闭通达信会话 |
| 行情数据 | K线、Tick、快照、订阅 |
| 股票信息 | 基础资料、股本、ETF跟踪 |
| 板块数据 | 板块列表、成分股、自选板块 |

| 可转债/新股 | 可转债信息、新股申购 |
| 交易日历 | 交易日列表 |
| 交易接口 | 账户、资产、委托、持仓、下单（默认dry-run） |
| 工具/缓存 | 公式执行、缓存刷新 |

## 使用门槛

- 需要有 **通达信/通达信量化** 终端（含策略能力）
- 配置策略路径（TQ_PATH）
- 新用户注册可领取 10,000 积分体验

## 与 WorkBuddy 集成

WorkBuddy 已官方接入通达信 MCP：
- 在左侧菜单「连接器」中找到「通达信链接」
- 一键授权安装
- 免费使用通达信金融数据

## 研究结论

**优点：**
- 打通了 AI 与实时 A 股数据的壁垒
- 支持自然语言查询，降低使用门槛
- 覆盖行情、技术面、基本面、资金面全维度数据
- 开源 + 官方双支持，生态成熟

**注意：**
- MCP 本身不能主动推送，需要自己写轮询+通知
- 交易接口默认 dry-run，需手动开启并自行承担风险
- 需要通达信终端或 TdxQuant 账号
