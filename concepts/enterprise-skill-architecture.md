# 企业级 Skill 体系

## 概述

企业最值钱的东西（流程、经验、数据处理规则）往往隐藏在系统和员工脑子里。Skill 的价值在于把这些经验"软件化"，让所有人都能用、AI 可调用。

## 核心概念对比

| 概念 | 说明 |
|------|------|
| SOP | 企业标准操作流程 |
| Skill | AI 可调用的能力模块 |
| Agent | LLM + Skills（龙虾装上各种技能）|
| MCP | AI 世界的 USB 协议，统一访问外部工具和系统 |

## 三者关系

- **LLM（大模型）** = 脑子，擅长理解语言和推理
- **Skill** = 手脚，给大模型提供执行能力
- **Agent** = LLM + Skill

## 未来企业软件形态

不再由 N 个独立软件系统组成，而是 Skills 组成的能力网络：

```
CRM Skill + ERP Skill + Finance Skill + HR Skill + BI Skill + ...
```

## 企业 Skill 体系三件套

| 组件 | 作用 |
|------|------|
| Skill Marketplace | 技能市场，分发和交易 Skills |
| Skill Library | 技能库，沉淀和管理企业能力 |
| Skill SDK | 技能开发工具包，开发和扩展 Skills |
