# GitHub AI Trending Projects

## LLM & Agents 平台

### 1. dify ⭐ 130,964 TypeScript
- **类型**：可视化 Agent 工作流开发平台
- **特点**：生产级、开箱即用
- **技术栈**：TypeScript
- **GitHub**：https://github.com/langgenius/dify

### 2. langchain ⭐ 127,977 Python
- **类型**：Agent 工程平台
- **特点**：行业标准框架
- **技术栈**：Python

### 3. ragflow ⭐ 74,053 Python
- **类型**：RAG（检索增强生成）引擎
- **特点**：企业级文档问答
- **技术栈**：Python

### 4. mem0 ⭐ 48,486 Python
- **类型**：AI Agent 通用记忆层
- **特点**：解决长期记忆问题

## AI 编程助手

### 5. anything-llm ⭐ 55,295 JavaScript
- **类型**：一体化 AI 应用
- **特点**：内置 RAG、Agents

### 6. continue ⭐ 31,607 TypeScript
- **类型**：开源 AI 编程助手
- **特点**：CI 集成、代码检查

## 模型训练与微调

### 7. LlamaFactory ⭐ 67,772 Python
- **类型**：统一微调框架
- **特点**：支持 100+ LLM/VLM
- **学术认可**：ACL 2024 论文

### 8. llama_index ⭐ 47,318 Python
- **类型**：文档 Agent & OCR 平台
- **特点**：数据索引专家

## AI 基础设施

### 9. marimo ⭐ 19,423 Python
- **类型**：响应式 Python notebook
- **特点**：可复现实验、SQL 查询

### 10. lance ⭐ 6,130 Rust
- **类型**：多模态 AI Lakehouse 格式
- **特点**：高性能向量存储

## 热点趋势总结

### 最火方向
1. **Agent 平台** - dify、langchain 领跑
2. **RAG 引擎** - ragflow、llama_index 主导
3. **AI 记忆** - mem0 专注长期记忆
4. **多模态** - lance、marimo 新兴
5. **AI 编程** - continue、anything-llm

### 技术栈趋势
- **TypeScript 在 AI 工具层崛起**（dify、continue）
- **Python 仍是模型训练首选**（langchain、ragflow）
- **Rust 进入基础设施层**（lance）
- **全栈方案受欢迎**（anything-llm）
