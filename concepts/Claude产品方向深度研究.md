## 一、整体产品架构概览

Anthropic 的产品分为两大体系：**模型层（Models）** 和 **产品应用层（Products）**。

```
                    ┌─────────────────────────────────────────┐
                    │              Claude 产品体系              │
                    └─────────────────────────────────────────┘
                                      │
              ┌───────────────────────┴───────────────────────┐
              │                    模型层                      │
              ├───────────────────────────────────────────────┤
              │  Haiku → Sonnet → Opus → Fable ──┬─→ Mythos  │
              │  (入门)   (日常)  (专业)  (旗舰)    │ (受限)   │
              │                                 Safeguards   │
              └───────────────────────────────────────────────┘
                                      │
              ┌───────────────────────┴───────────────────────┐
              │                   产品应用层                   │
              ├───────────────────────────────────────────────┤
              │  Claude.ai (对话) │ Claude Code (编码)         │
              │  Claude Cowork (知识工作) │ Claude Security     │
              │  Claude Platform (API) │ Enterprise             │
              │  插件: Slack / Microsoft 365 / Chrome          │
              └───────────────────────────────────────────────┘
```

## 二、模型产品详解

### 2.1 Claude Haiku 4.5 — 极速轻量

**定位：** 最快、最便宜的模型，适合高并发、低延迟场景

**价格：** $1/M 输入 tokens，$5/M 输出 tokens

**核心特点：**
- 运行速度是 Sonnet 4.5 的 4-5 倍
- 在 SWE-bench Verified 上得分 73.3%，匹配 Sonnet 4 的编程能力
- 性价比极高，模糊了"轻量模型"与"前沿模型"的边界

**核心场景：**

| 场景 | 价值分析 |
|------|---------|
| 免费用户体验 | 以极低成本为所有用户提供 AI 能力，适合产品免费层 |
| 实时客服/聊天 | 毫秒级响应，适合对延迟敏感的客户交互 |
| 编程子代理 | 多代理系统中的执行节点，负责快速简单任务 |
| 金融数据监控 | 并行处理数千个数据流，实时追踪风险和监管变化 |
| 规模化研究 | 同时分析数十个文献来源，几小时完成数周工作量 |

**战略价值：** Haiku 是 Anthropic 的"规模化引擎"——让 AI 从高端专业工具变成普惠基础设施。

---

### 2.2 Claude Sonnet 4.6 — 旗舰日常模型

**定位：** 性能与成本的最优平衡，Claude 的"主力出货模型"

**价格：** $3/M 输入 tokens，$15/M 输出 tokens（Haiku 的 3 倍，Opus 的 60%）

**核心特点：**
- 首个混合推理模型（Hybrid Reasoning），可切换标准回答与深度思考模式
- 1M token 上下文窗口（API Beta）
- 支持 extended thinking，可控制思维深度
- 特别强化了 coding、agentic workflows、computer use 能力

**核心场景：**

| 场景 | 价值分析 |
|------|---------|
| 高级编程 | 全生命周期覆盖：规划→实现→调试→重构，多文件协同编辑 |
| 长时运行代理 | 自主处理复杂多步骤任务，适合客服自动化、内部工作流 |
| 浏览器/计算机使用 | 自动化基于浏览器的业务流程（竞情分析、采购、入职等） |
| 企业级工作流 | 金融分析、内容创作、商业运营的高容量任务处理 |
| 创意写作 | 保持多角色一致性，适合叙事性内容生成 |

**战略价值：** Sonnet 是 Anthropic 的"现金牛"——足够智能、价格适中、速度够快，是大多数企业级应用的默认选择。

---

### 2.3 Claude Opus 4.8 — 专业级旗舰

**定位：** 高级专业工作的高可靠模型，面向对质量要求极高的人群

**价格：** $5/M 输入 tokens，$25/M 输出 tokens（Haiku 的 5 倍）

**核心特点：**
- 自适应思考（Adaptive Thinking）——自动根据任务复杂度调整思维投入
- 强化了严肃编程（SWE）和 AI 代理的可靠性
- 在 CursorBench、Legal Agent Benchmark、Online-Mind2Web 等多项基准测试中刷新纪录
- Opus 4.8 是唯一在 Super-Agent 基准上完成所有端到端用例的模型

**核心场景：**

| 场景 | 价值分析 |
|------|---------|
| 高级编程 | 可委托完整的生产级代码任务，减少资深工程师的重复劳动 |
| AI 代理生产环境 | 长时间、多工具协同的自动化工作流驱动引擎 |
| 企业工作流 | 跨会话记忆、多日项目追踪，专业文档/表格/演示交付 |
| 法律专业工作 | 首个在 Legal Agent Benchmark 突破 10% all-pass 的模型 |
| 金融文档分析 | 精确引用、多文档推理，适合密集型 filing 分析 |

**战略价值：** Opus 是 Anthropic 的"品质标杆"——定价高、面向专业市场，树立行业性能天花板。

---

### 2.4 Claude Fable 5 — 次世代受限旗舰

**定位：** 与 Mythos 5 同源，添加了网络安全和生物学的安全保障机制后公开发布

**价格：** $10/M 输入 tokens，$50/M 输出 tokens（Opus 的 2 倍）

**⚠️ 当前状态：** 因美国政府出口管制指令，Fable 5 和 Mythos 5 已于 2026-06-12 暂停访问

**核心特点：**
- Fable 5 = Mythos 5 + 健壮的安全护栏
- 在网络安全和生物学领域会自动降级到 Opus 4.8
- 支持数天的异步长周期任务
- 可编写自己的测试用例来验证输出

**核心场景：**

| 场景 | 价值分析 |
|------|---------|
| 多日代理 | 在 Claude Code / Managed Agents 中可连续运行数天 |
| 雄心勃勃的编码项目 | 大规模迁移、复杂实现、跨月自主编程会话 |
| 深度企业研究 | 多阶段知识工作，从研究到可交付成果的完整流程 |
| Vibe Coding | 在 ViBench 基准上接近饱和，构建应用的速度和准确度最高 |
| 前沿代码迁移 | 50 亿行 Ruby 代码库中，1 天完成原本需要 2 个月的工作 |

**战略价值：** Fable 代表了 Anthropic 的技术巅峰——把 Mythos 的全部智能释放给大众，同时通过护栏管控风险。

---

### 2.5 Claude Mythos 5 — 受限前沿研究模型

**定位：** Anthropic 最高能力的模型，专注于网络安全和生物研究，仅限审核合作伙伴

**价格：** $10/M 输入 tokens，$50/M 输出 tokens

**⚠️ 当前状态：** 因美国政府出口管制指令，Fable 5 和 Mythos 5 已于 2026-06-12 暂停访问

**核心特点：**
- 在网络安全、生物学、医疗领域的最高能力模型
- 目前仅通过"信任访问计划"向少量合作伙伴开放
- 与 Fable 共享同一底层模型，但无安全护栏限制
- 30 天数据保留政策（用于安全监控）

**核心场景：**

| 场景 | 价值分析 |
|------|---------|
| 网络安全研究 | 漏洞发现（Project Glasswing 已发现 10,000+ 高危漏洞） |
| 生物学研究 | 药物发现、蛋白质分析、公共卫生研究 |
| 前沿 AI 安全研究 | 与顶级科技公司（Microsoft/NVIDIA/Google/Apple 等）合作的 Glasswing 项目 |

**Project Glasswing 生态：**
> 参与方：AWS、Anthropic、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorganChase、Linux Foundation、Microsoft、NVIDIA、Palo Alto Networks

## 三、产品应用层详解

### 3.1 Claude Code — AI 编程代理

**产品形态：**
- 终端工具（CLI）
- VS Code 插件
- JetBrains 插件
- GitHub Actions 集成
- 移动端→桌面端任务接力（从手机描述任务，回来后看到 PR）

**核心能力矩阵：**

| 功能 | 描述 |
|------|------|
| 代码库理解 | agentic search 自动理解整个代码库结构，无需手动选上下文 |
| PR 全流程 | 读 Issue → 写代码 → 运行测试 → 提交 PR，一条命令完成 |
| 多文件协同编辑 | 理解依赖关系，跨文件精准修改 |
| 终端内工作 | 不离开终端，覆盖编码→部署→监控全流程 |
| 代码审查 | 自动分析 PR，提出改进建议 |

**定价：** 已包含在 Pro/Max 订阅中；Enterprise 独立定价

**典型用户证言：**
- Notion：用 Claude Code 处理 Metaflow 管道搭建，节省 1-2 天/模型
- Ramp：EDA 代码到生产管道的自动化

**战略价值：** Claude Code 是 Anthropic 进入软件开发工作流的核心入口，定位是"AI 结对编程伙伴"而非简单的代码补全。

---

### 3.2 Claude Cowork — 知识工作 AI

**产品形态：** Web/移动端，Claude Code 的能力迁移到知识工作场景

**核心定位：** 用 AI 代理的方式处理知识密集型工作（不是对话，是委托任务→等待结果）

**典型场景：**
- 长篇报告研究
- 多文档综合分析
- 任务自动化（类似 Claude Code 但面向非代码工作）

---

### 3.3 Claude Security — 安全 AI 产品

**产品定位：** 专注于安全领域的企业级 AI 产品

**已知信息：**
- 从 Mythos 模型继承的安全研究能力
- 与 Project Glasswing 深度整合
- 针对网络安全团队的工作流优化

---

### 3.4 Claude Platform — API 平台

**核心功能：**
- 原生 API 访问所有 Claude 模型
- Prompt Caching（最多节省 90% 成本）
- Batch Processing（节省 50% 成本）
- 数据驻留选项（US-only 推理 1.1x 定价）
- 合作伙伴：AWS Bedrock、Google Vertex AI、Microsoft Foundry

---

### 3.5 企业级产品

**Claude Code Enterprise：**
- 企业级安全与合规
- SSO / SCIM 集成
- 私有部署选项
- 使用管理与审计

**定价体系：**

| 层级 | 目标用户 | 主要包含 |

| Free | 个人用户 | Haiku |

| Enterprise | 企业 | 自定义定价，私有部署 |

## 五、战略分析：Anthropic 的产品哲学

### 5.1 纵向分层：能力×价格矩阵

Anthropic 没有只做一个"最强模型"，而是构建了一个**能力谱系**：
- **Haiku** = 扩大用户基数，降低 AI 使用门槛
- **Sonnet** = 商业化主力，覆盖 80% 真实用例
- **Opus** = 树立专业市场品质标杆
- **Fable/Mythos** = 技术前沿探索，同时承担 AI Safety 的商业化平衡

### 5.2 横向扩展：从模型到工作流

Anthropic 的野心不止于"更好的 LLM"，而是**重新定义知识工作方式**：
- Claude Code → 重新定义软件开发
- Claude Cowork → 重新定义白领知识工作
- Claude Security → 重新定义网络安全

### 5.3 安全即壁垒

Mythos/Fable 的双轨制（无护栏研究版 vs 有护栏商业版）是 Anthropic 的独特商业模式：
- Mythos 面向可信合作伙伴，收取研究溢价
- Fable 面向大众市场，用护栏换规模

### 5.4 合作伙伴生态

从 Project Glasswing 可以看出，Anthropic 选择与整个行业抱团：
- AWS、Google、Microsoft 云端部署保证分发
- Apple、NVIDIA 硬件级合作
- JPMorganChase、Microsoft、CrowdStrike 垂直场景深耕

## 六、关键事件：Fable 5 & Mythos 5 出口管制

**2026-06-12：** 美国政府发布出口管制指令，暂停 Fable 5 和 Mythos 5 的所有访问。

**影响：**

- 此事揭示了前沿 AI 模型的战略敏感性

## 七、总结：产品价值地图

```
        价值创造
           ↑
           │    Mythos 5 (前沿研究)
           │    Fable 5 (旗舰商业)
           │
           │    Opus 4.8 (专业高可靠)
           │
           │    Sonnet 4.6 (商业主力)
           │
           │    Haiku 4.5 (规模普及)
           │
           └──────────────────────────────────→ 场景复杂度
                Coder    Enterprise   Research
                 ↑          ↑           ↑
              Claude    Claude      Claude
              Code     Cowork     Security
```

**核心洞察：** Anthropic 的产品矩阵揭示了一个清晰的战略——不只做"更聪明的 AI"，而是通过**分层模型×垂直产品×行业生态**的三重布局，成为 AI 时代的基础设施提供商。
