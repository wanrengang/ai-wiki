# Xiaomi miclaw

## 概述

Xiaomi miclaw（昵称"龙虾"）是小米于 2026 年 3 月 6 日正式发布的移动端 AI 智能体（Agent）产品，基于小米自研的 MiMo 大模型构建。是**国内首个手机端类 OpenClaw Agent 应用**。

核心定位：系统级 AI 智能体，以系统应用身份运行，拥有调用系统核心能力和生态服务的权限。

## 核心能力架构

### 2.1 系统与生态调用
- 已封装 **50+ 项系统能力和生态服务**（持续扩展）
- 用户授权后可调用一方应用、系统工具及米家 IoT 设备
- 可执行多步骤复杂操作

### 2.2 个性化记忆与理解
采用**三级智能记忆管理**：
- 自动保留关键决策点
- 动态压缩冗余交互
- 核心指令本地缓存
- 即使执行 20 步复杂操作，也能连贯理解初始意图

### 2.3 主动服务与决策
- 分析银行扣费短信，发现重复订阅后推送节省开支建议
- 根据日历事件自动将手机静音、暂停扫地机器人
- 主动感知场景并提供服务

### 2.4 持续成长能力
作为"创造工具的工具"，具备四种能力：
- 文件级记忆创造
- 子智能体创建
- MCP 服务配置
- 脚本执行

## 与传统 AI 助手的区别

| 维度 | 传统 AI 助手 | Xiaomi miclaw |
|------|------------|---------------|
| **运行方式** | 应用层语音助手 | 系统应用身份运行 |
| **能力调用** | 有限 API 调用 | 50+ 项系统能力 + 米家生态 |
| **记忆能力** | 会话级记忆 | 三级智能记忆管理 |
| **执行能力** | 单轮问答 | 多步骤复杂任务执行 |
| **主动性** | 被动响应 | 主动服务与决策 |
