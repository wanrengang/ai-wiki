机器之心编辑部 2026 年 5 月的第一周，有三件事几乎同时发生在同一门生意上。

5 月 1 日，孙宇晨在 X 上发帖：「史上最强 AI 中转站 B.AI 白来了。一个 API Key = Claude + GPT + Gemini + 国产大模型全系列。区块链登录，纯匿名支付，零篡改，价格全网最低。」这位去过太空的男人，亲自带货了。

猎豹移动 CEO 傅盛也宣布推出 Easy Router，口号是「全场 8.5 折」，声称一个 Key 调用 40 余个主流大模型，零平台费、零加价。

5 月 5 日，与特朗普家族深度绑定的加密项目 WLFI 转发了一个名为 WorldClaw 的新产品。它的最高套餐售价 9999 美元，买了之后你将获得 100 万 AI 积分、一台硬件设备，以及一张有机会参加海湖庄园私人晚宴、与小特朗普共进晚餐的抽签券。

同一周，同一赛道，三个极不相似的名字。知乎上「为什么特朗普、孙宇晨、傅盛纷纷入局 AI 中转站卖 Token 了？」的问题浏览量迅速超过 75 万次。

AI 中转站，这个被人叫作「AI 二道贩子」或「Token 进口商」的灰色生意，突然站上了公众视野的正中央。原因很简单：它足够赚钱。

这门生意，并不新鲜

要理解 AI 中转站，先回想 2000 年前后的互联网。那个年代，最赚钱的不是雅虎或搜狐这样的门户网站，而是那些不太起眼的人：拉光纤的、卖服务器机架的、做 IDC 托管的。内容是花瓶，基础设施才是印钞机。AI 中转站在 2024、2025 年扮演的，正是当年那批人的角色。

它的技术本质是架在用户和 AI 模型厂商之间的一层反向代理。用户把请求发给中转站，中转站转发给 OpenAI 或 Anthropic 的服务器，取回结果后再交给用户。听起来不过是一层中间人，却因为几个现实条件，成了一门真正能赚钱的生意。

第一个条件是价格错位，而且错位幅度大得离谱。以 OpenAI 为例：订阅 Pro 套餐，200 美元一个月，包含大量的使用额度。但如果你直接走 API 按量付费，同等调用量折算下来可能高达每月数千美元，差距百倍级别。Claude 的情况更极端：以 Claude Max 订阅账号为例，月费约 200 美元，但其可承载的 API 调用量，若按官方 API 定价折算，价值可达 400 至 600 美元，中间存在非常显著的差价利润。

第二个条件是访问壁垒，针对的是中国和全球南方用户。对国内开发者来说，直接购买海外 AI 模型的 API 面临三重障碍：支付链路（没有绑定境外银行卡的渠道）、网络访问（需要稳定的代理环境）、以及企业合规（无法获取正规发票入账）。大多数小型 AI 应用公司，与其自建账号池维护运维，不如直接向中转站采购，后者甚至可以提供对公发票。对更广泛的全球南方用户而言，没有 Visa 卡、没有美区账号、银行通道被封堵，意味着他们被官方彻底挡在最先进的 AI 工具门外。这批人的需求，是真实的，也是长期被忽视的。

第三个条件是技术门槛极低，低到让入场成本几乎可以忽略不计。开源社区在过去两年为这个行业贡献了近乎完整的基础设施。One API（GitHub 超 3.3 万星，Go 语言编写）支持 OpenAI、Claude、Gemini、DeepSeek 等几乎所有主流模型，全量兼容 OpenAI 标准 API 格式，一行 Docker 命令即可部署，带上数据库和缓存也不过十几行配置。在此基础上二次开发的 New API，则提供了更完善的商业化功能，成为众多中转站事实上的底层框架。

市场规模随之快速生长。据行业目录 TokenNav 统计，截至 2026 年，国内外至少有 92 个类似产品在运营。从业者在公开论坛的讨论中普遍认为头部项目盈利可观。

三个人，三门不同的生意

如果你只看表面，傅盛、孙宇晨和特朗普家族在做同一件事：卖 AI Token 的中转代理。但只要往深挖一点，就会发现他们的动机、逻辑和目标用户几乎毫无交集。用一句话概括：傅盛卖便利，孙宇晨卖管道，特朗普家族卖门票。

傅盛：上市公司需要一个新故事

猎豹移动曾经是中国互联网清除流氓软件的代表。在 PC 红利消退后，傅盛带领猎豹转型过机器人、转型过出海工具，但都未能打出真正的第二曲线。这次 EasyRouter 的出现，在逻辑上很清晰：作为 Google Cloud 和 AWS 的认证合作伙伴，猎豹移动在云端采购上具备渠道折扣优势，可以将部分优惠传导给用户。傅盛在后来的说明中也强调了这一点。全场 8.5 折不是噱头，是渠道差价的一部分回馈。

孙宇晨：把 TRON 塞进 AI 的每一笔微支付

孙宇晨的 B.AI，外表看上去是中转站，内里是一套为 AI Agent 时代搭建的支付基础设施。B.AI 的核心卖点不是折扣，而是「无身份」：TRON 链钱包登录，USDT 结算，无需注册账号，无需信用卡，无需 KYC（身份核验）。这直接打通了两类巨大的需求：一是无法通过常规渠道支付的全球南方用户，二是需要在链上自主完成任务和支付的 AI Agent。截至 2026 年 4 月，TRON 网络已积累逾 3.76 亿个账户，承载着超过 860 亿美元的 USDT 流通量，每日处理转账额逾 230 亿美元。这个体量意味着 B.AI 并不需要自己去构建支付网络——TRON 已经是一个成熟的全球稳定币结算层，B.AI 只是在这个管道上接入了 AI 模型这个新的消费场景。

故事的戏剧性在于孙宇晨的处境。就在 B.AI 发布的同一时期，他刚刚于 4 月 21 日在加利福尼亚州联邦地区法院以欺诈、违约和不当得利起诉 WLFI（特朗普家族的加密项目），指控对方通过智能合约中隐藏的黑名单功能冻结了他持有的约 29 亿枚代币——这批持仓在冻结时价值逾 1 亿美元，随 WLFI 价格下跌，至起诉时已缩水至约 7500 万美元。WLFI 则在 5 月 4 日反手在佛罗里达州迈阿密-戴德县提起诽谤诉讼。两边打着官司，5 月 5 日，WLFI 的 WorldClaw 上线了——和 B.AI 几乎一模一样的产品形态。

特朗普家族：AI 只是新包装纸，他割的还是那批韭菜

WorldClaw 由 WLFI 生态孵化，定位为「AI Agent 操作系统」，核心产品 WorldRouter 接入了超过 300 个 AI 模型，号称价格比官方低 30%。唐纳德·特朗普 Jr. 和埃里克·特朗普均在社交媒体公开转发了产品发布公告。

表面上，这是一个 AI 中转站。但仔细审视其收费设计，会发现一个精心构建的四层收割结构：

第一层：API 调用差价。这是最表层的收入，也是最薄的一层。

第二层：用户必须用 USD1 支付，这是 WLFI 于 2025 年 3 月发行的稳定币。每一笔 AI 消费，都在为 USD1 制造真实的使用场景和流通需求。WLFI 已申请全国银行信托牌照，野心指向的是「把 USD1 做成合规的下一代 USDT」。

第三层：用户可以选择锁仓 WLFI 代币来换取 AI 积分（Pro 档需锁 250,000 枚，Max 档需锁 2,500,000 枚）。大量代币被锁定，减少了市场流通量，推升代币市值，WLFI 团队持有的大量早期代币随之升值。

第四层：$9999 的 Max 套餐中附带一张「有机会」参加海湖庄园私人晚宴的抽签券。这是营销噱头，更是美国政治准入权的定价变现。研究政治献金的学者早已清楚，直接接触权力圈层是有市场价格的。9999 美元这个数字，恰好处于美国合规政治献金的某个门槛附近。

买 API 他收一次，换 USD1 他收一次，锁 WLFI 他收一次，晚宴抽签他收一次。一个 API 调用，他能割四刀。

这门生意里，你不知道的那些黑

三个名人的入场，让 AI 中转站从技术圈走进了大众视野。但喧嚣的背后，这个行业长期存在一套系统性的用户伤害机制，鲜少被公开讨论。

你付了 GPT-5 的钱，收到的可能是 Llama

在中转站行业，「注水」（模型替换）是公开的秘密。由于 AI 模型输出的质量差异对大多数普通用户来说难以肉眼识别，把廉价模型冒充高端模型，成了利润最稳定、风险最低的套利方式。

2026 年 ACM 互联网测量大会上发表的论文《Behavioral Consistency and Transparency Analysis on Large Language Model API Gateways》，对真实商业网关进行了系统性测评。结果显示：逾四成被测端点的模型指纹验证失败，即你调用的模型和平台声称提供的模型并非同一个；在实际任务表现上，偏差最高达到 47%。更讽刺的是，价格高低对结果准确性完全没有预测力——花更多钱，并不能让你免受替换。

替换手法从粗糙到精密。最低端的是「概率降级」：50 次请求里抽 7、8 次转到便宜渠道，质量波动明显、Token 计数对不上，容易被发现。高端版本是「版本静默切换」：每次模型迭代，中转站把后端悄悄换成更便宜的旧版本，用户浑然不觉。
